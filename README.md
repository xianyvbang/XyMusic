## 支持服务及版本

- [x] Jellyfin 10.10.7及以上
- [x] Emby 4.1.1.0及以上
- [x] Subsonic 1.16.0及以上
- [x] Navidrome 0.56.1及以上
- [x] Plex 4.1.1.0及以上
